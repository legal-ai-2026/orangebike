Connecting Tableau

Option 1: CSV Import (Recommended)

  1. Click "Download All as ZIP" above
  2. Unzip the file on your computer
  3. In Tableau: Connect > Text File
  4. Select any CSV to start, then drag others onto the canvas to create joins
  5. Join wholesale_orders to accounts on account_name
  6. Join fact tables to beer_styles on style_name

Option 2: Direct SQLite

  1. In Tableau Desktop: Connect > SQLite
  2. Navigate to:
    Data/orange_bike.db
  3. All tables appear automatically
  4. Drag tables onto the canvas and Tableau will detect relationships

Note: Tableau Public does not support SQLite — use CSV import instead.
